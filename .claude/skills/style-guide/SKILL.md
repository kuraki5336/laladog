當修改或建立 UI 元件時，必須嚴格遵循以下視覺規範。

完整規格請參考 [reference.md](reference.md)。

## Quick Reference

- 中文字體：`Noto Sans TC`
- 英文字體：`Rounded Mplus 1c`（Regular）/ `Rounded Mplus 1c Bold`（Bold）
- Primary 主色：`#00416A`（Primary 60）— 深藍色系
- Secondary 次要色：`#3298DC`（Secondary 50）— 亮藍色系
- 全站背景：`#F4FAFD`
- 文字主色：`#475569`（Slate）
- 文字次色：`#64748B`（Slate 500）
- 文字輔色：`#94A3B8`（Slate 400）
- 功能色：Success `#4CAF50` / Warning `#FFC107` / Danger `#E63946` / Info `#3298DC`
